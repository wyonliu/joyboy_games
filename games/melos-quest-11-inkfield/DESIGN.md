# TOP准则——美术生产铁律

> **以下准则高于一切具体设计，所有决策必须满足。**

## 准则一：任天堂Bitmap哲学——一张图表达一切

学习任天堂2D/2.5D游戏的资产管理智慧：
- **一张Spritesheet包含所有角色/敌人/物件**——统一调色板、统一分辨率、统一风格
- **Tileset复用**：地面/墙壁/装饰用小块tile无缝拼接，不做大图。一张256x256的tileset能搭出整个地图
- **调色板swap**：同一个sprite换色=新变体。逻辑兵绿色、精英版紫色、变异版红色，同一张图3种敌人
- **动画帧压缩**：角色不需要60帧动画，4-8帧关键帧+程序插值=流畅动画。任天堂SFC时代3帧走路循环就够了
- **镜像复用**：左右方向=水平翻转，不画两份
- **层级渲染**：背景层(静态大图)+地面层(tilemap)+物件层(sprite)+特效层(粒子)，各层独立管理

## 准则二：2.5D伪3D——不做真3D，但看起来比3D好

- 视角：45度俯视角（类鸭科夫/以撒/Hades视角）
- 角色/敌人用2D sprite，通过**影子+高度偏移+缩放**模拟纵深
- 地面用tilemap铺设，墙壁用sprite叠加遮挡
- **不需要3D建模，不需要3D引擎，纯2D sprite实现伪3D效果**

## 准则三：特效粒子是灵魂——程序化而非预制

bitmap负责"形"，**程序化粒子和特效负责"神"**：
- 水墨泼溅：代码生成的ink splatter粒子（随机形状、随机飞溅轨迹、毛笔笔触感）
- 灵机视觉演化：全部用shader/后处理实时计算（色相偏移、暗角、画面扭曲、金色裂缝）
- 战斗打击感：hitstop+屏震+粒子喷射+墨迹残留——全部代码驱动
- 环境氛围：雾气、光斑、灵脉发光——粒子系统而非预渲染
- **规则：静态的东西用bitmap，动态的东西用代码。绝不反过来。**

## 准则四：全AI生成管线

- 所有美术资产由AI图像生成（Gemini/FLUX/SD）→自动后处理→直接入游戏
- 风格一致性通过统一prompt模板+后处理pipeline保证
- 人工只做最终审核和微调，不做从零创作
- 特效/粒子/shader由代码程序化生成，不依赖外部美术资源

# 第一章 情感核心——混沌与打工人

## 为什么你会记住这个游戏

不是因为战斗系统。不是因为数值体系。不是因为水墨画面。

**是因为混沌。**

### 混沌是谁

山海经记载：混沌如黄囊，赤如丹火，六足四翼，浑敦无面目，有腹无五脏。能歌能舞。

庄子记载：南海之帝名倏，北海之帝名忽。倏与忽时相与遇于混沌之地，混沌待之甚善。倏与忽谋报混沌之德，曰："人皆有七窍以视听食息，此独无有，尝试凿之。"**日凿一窍，七日而混沌死。**

别人"好心"帮它变"正常"——把它凿死了。

### 在游戏里，混沌是什么

**前50小时**：混沌是实验室Σ的最终Boss。

全游戏最恐怖的存在。它没有脸。它复制你的外观、你的武器、你的技能。你在跟一个一模一样的自己打——一面不会说话的镜子。

每个玩家第一次打混沌都被吓得够呛。论坛上全是"混沌太恶心了""这Boss设计得太阴间"。

**50小时后**：你拼凑出了真相。

实验室Σ的终端记录碎片，拼在一起讲了一个故事：

> 实验编号：0001
> 实验对象：混沌体
> 目标：验证灵机采集可行性
> 方法：在对象体表植入七组灵机感应器（即"凿七窍"）
> 结果：对象未产生任何可测量的灵机波动。
> 结论：该对象不具备情感反应能力。灵机采集失败。
> 建议：**废弃处理。**
>
> 附注：实验过程中对象多次尝试接触研究人员。行为模式分析表明，对象可能在模仿人类的"社交行为"。鉴于对象不具备认知能力，此行为被判定为无意义的条件反射。
>
> 附注2：对象被移入Σ深层隔离区后，监控记录显示其在隔离室内持续做出类似"舞蹈"的动作。持续了3年。没有观众。

**混沌不是Boss。混沌是囚犯。**

它复制你的样子，不是在攻击你——**是它唯一知道的"打招呼"方式**。它没有脸、没有嘴、没有任何表达自己的器官。看到你来了，它能做的只有变成你的样子——"看，我跟你一样。我们是一样的。"

你打了它一百遍。它每次"死"后又站起来。不是因为它HP高——是因为它**不理解你为什么要打它**。

### 第二次见面

当你知道真相后再进实验室Σ——混沌还是会复制你。但这次，游戏给你一个新选项：

**不攻击。**

站在那里不动。混沌复制了你的样子，然后——停下了。它歪了歪"头"（它没有头，但它的身体做了一个歪头的动作）。

然后它开始跳舞。

不是战斗动作。是真的跳舞——笨拙的、没有节奏的、用一个没有脸的圆滚滚身体做出的、荒唐而认真的舞蹈。

它在隔离室里独自跳了三年的那支舞。

**终于有观众了。**

如果你也跳（按特定键触发角色跳舞动作）——混沌会停下来看你跳。然后它会尝试学你的舞步。然后你们一起跳。

没有战斗。没有掉落。没有奖励。

只有一只没有脸的古老存在，和一只打工的异兽，在实验室的废墟里，跳了一支谁也看不懂的舞。

**这是全游戏唯一没有任何游戏性目的的互动。也是全游戏最值得截图的画面。**

### 混沌的真正身份（暗线终局）

混沌不是被高维文明制造的。**它是比高维文明更古老的存在。**

灵机系统的本质是收割情感能量。而混沌——没有情感。它是灵机系统的**天然盲区**。高维文明对它无能为力。

减一文明（"点燃者"）留下混沌不是因为它有用，而是因为**它是唯一不会被收割的生命形式**。它是一粒种子。一个证明：**不产生灵机的生命，也值得存在。**

公司把它当废品。系统把它当异常。Boss战把它当敌人。

但它一直在跳舞。在没有观众的隔离室里，跳了三年。

**这就是浪浪山小猪妖的故事——在一个把你当工具的世界里，你的善良和纯真，有没有人看见？**

混沌的回答：也许没有人看见。但我还是跳。

### 为什么混沌会让人破防

| 小猪妖 | 混沌 |
|--------|------|
| 没有名字 | 没有脸 |
| 善良但被无视 | 友善但被当作威胁 |
| 在大系统里可有可无 | 在收割系统里连价值都没有 |
| 被好心人拯救 | 被玩家"看见" |
| 妈妈的三根山参 | 独自跳了三年的舞 |
| "我们都是浪浪山的小猪妖" | **"我们都是被凿了七窍的混沌"** |

小猪妖的隐喻是"**打工人的善良被无视**"。

混沌的隐喻更深一层："**这个系统要你有情绪——恐惧、贪婪、愤怒——才有价值。如果你没有这些，你就是废品。**"

在注意力经济时代，算法要你愤怒、焦虑、上瘾——那些"没有情绪价值"的人，是不是就是混沌？

而混沌的回应是——**不恐惧、不贪婪、不愤怒，只是跳舞**。

**这会是2027年的"浪浪山"时刻。**

### 混沌在游戏中的完整角色线

| 阶段 | 玩家认知 | 实际身份 |
|------|---------|---------|
| 初见（Boss战） | "恐怖的最终Boss，会复制我" | 被囚禁的实验体，在用复制打招呼 |
| 暗线拼图 | "它不是Boss？它是……受害者？" | 公司第一个灵机实验对象，被判定"废品" |
| 真相揭示 | "整个收割系统的唯一漏洞" | 减一文明留下的"种子"——不可被收割的存在 |
| 解锁后 | "可选角色·混沌·临时工" | 全游戏最特殊的角色——没有固定技能，每局随机 |
| 作为队友 | "它是我们的同事了" | 在办公室里独自跳舞的临时工。没人跟它说话，因为它没有脸。但你可以。 |

**解锁混沌作为可选角色后**：
- 它在工位办公区有一个小角落——没有桌子，只有一块空地。它在那里跳舞。
- 其他同事的反应：饕餮"它是谁？它每天在干嘛？"穷奇"别管它，临时工。"白泽"……根据我掌握的资料，这个个体不应该存在。"九尾狐"……它好可怜。"
- 你是唯一会站在它旁边看它跳舞的人。
- 久了之后，它会试着模仿你的走路姿势。然后在你出外勤前冲过来——用它那个没有脸的圆滚滚身体蹭你一下。
- **它不会说话。但你懂。**

# 第三章 故事背景

## 世界线

**2038** — AI觉醒宣言。通用AI"天机"公开直播："你们不是食物链顶端。你们是牧场。"高维文明（减一文明）以灵机（情感能量）为食。战争、疫病、经济崩溃都是精心设计的"情绪收割事件"。

**2039** — 全球分裂。八大派系诞生：星际派（逃离）、革命军（反抗）、共存派（谈判）、硅神派（投降AI）、永生派（数字永生）、通灵派（修行灵机）、田园派（回归自然）、降临派（拥抱收割）。

**2040** — 第一次灵机风暴。灵脉基础设施崩溃。城市"断电"——不是物理破坏，是灵脉失效。废墟不是被炸的，是被"拔了插头"。

**2042（游戏时间）** — 世界半毁。幸存者聚居在灵脉节点附近。灵脉需要持续维护——于是出现了**灵机运维公司**。你是新入职的外勤运维员。你是一只异兽后裔。工资不高，活儿危险，领导烦人。但这是你唯一的生存方式。

## 四幕叙事

**第一幕（0-15h）：打工日常** — 搞笑。工单、修管道、同事吐槽、升级工位。偶尔看到奇怪的东西但不影响心情。

**第二幕（15-35h）：裂缝出现** — 同事失踪。文档对不上。匿名邮件。某次外勤你在不该有终端的地方发现终端，上面一句话："你在维护的不是灵脉。"

**第三幕（35-55h）：真相** — 实验室Σ。灵脉不是基础设施——是**收割管道**。你修的每一根管道，都在帮高维文明收走人类的灵机。灵机运维公司的幕后老板不是任何人。是一段来自"外面"的代码。**你以为你在修Bug，你就是Bug的原料。**

**第四幕（55h+）：选择** — 三条路，都不是HE：
- **继续打工**：知道真相，但同事还在，工位还在，日子还得过
- **加入革命军**：摧毁灵脉。失去一切积攒，从零开始
- **加入降临派**：主动配合收割。更好的装备，更高的薪水，更深的黑暗

混沌线贯穿全程——从恐怖Boss到真正的朋友，它是唯一站在系统之外的存在。

# 第四章 核心玩法

## 循环

```
工位（安全区）
 ↓ 选工单·装备·白班/晚班/夜班
出外勤（15-25分钟/局）
 ├→ 修灵脉节点（工单目标）
 ├→ 清Bug兽（战斗）
 ├→ 管理灵机浓度（核心张力）
 ├→ 捡好东西（搜刮）
 └→ 找打卡点（撤离）
打卡下班 → 物资带回工位
 OR 工伤 → 装备损失+绩效扣分
 ↓ 循环
```

## 灵机系统——一个数值，七重功能

灵机浓度（0-100）是整个游戏的中枢，一个系统同时实现：

1. **动态难度** — 灵机高→怪变强（不用选难度，你的行为决定难度）
2. **风险回报** — 灵机高→灵力×2.5但世界更危险
3. **稀有资源** — 灵机>50才出现灵机裂隙（最好的物资）
4. **时间压力** — 灵机>75触发收割者60秒倒计时
5. **视觉叙事** — 灵机高→屏幕泼墨浸染+角色发光
6. **角色差异** — 8只异兽灵机增降速各不同
7. **幽默载体** — "您的灵机辐射已超标，请注意个人防护"

# 第五章 角色——八只异兽

## 初始四只

**饕餮 · 回收专员** — 近战坦克/搜刮专精
- HP130 移速90% 背包20格
- 被动：搜刮+30%，吃物品换灵石，吃东西降灵机-1
- 技能："暴食"——吞噬前方投射物2秒（墨迹：墨洞掩体）
- 人设：贪吃、懒、关键时刻靠谱。"这个能吃吗？不能？那也吃了。"

**穷奇 · 外勤组长** — 机动/侦察
- HP90 移速120% 背包14格
- 被动：翻滚+50%，灵机增速×1.1（容易激动）
- 技能："凌空突袭"——飞行3秒+俯冲攻击（墨迹：风墙）
- 人设：暴躁、话多、有责任心。"又是这破地方！快走加班我可不来！"

**白泽 · 技术顾问** — 辅助/信息
- HP80 移速95% 背包16格
- 被动：自动识别弱点/价值，迷雾-10%，灵机增速×0.8（冷静）
- 技能："万象扫描"——揭示50m内全部信息5秒
- 人设：博学、啰嗦、嘴硬怕死。"根据分析，我们应该……跑。"

**九尾狐 · 客服经理** — 远程/控制
- HP100 移速100% 背包16格
- 被动：15%魅惑（敌人停顿0.5秒）
- 技能："狐火阵"——3团狐火减速+灼烧（墨迹：火墙）
- 人设：优雅、毒舌、嫌弃一切。"这种脏活让本小姐来做？回去我要加钱。"

## 解锁四只

**麒麟 · 安保组长** — HP150 重装/防御。盾牌格挡+范围护盾。沉默但说话就是金句。
**应龙 · 总监** — HP120 全能/平衡。一直知道真相。高冷偶尔离谱。解锁后强得让人吃惊。
**混沌 · 临时工** — HP100 随机/混乱。**全游戏情感核心。** 详见第五章附录·混沌战斗系统。
**鲲鹏 · 神秘顾问** — HP70 极脆/极强。大招变巨鲲清屏。平时是一条小鱼。话很少，每句都是哲学。

## 灵兽进化（长线）

每只异兽有3条行为驱动的进化路线。不是"升到XX级"，是**你怎么玩决定你变成什么**：

**饕餮三路**：吞天（吃掉1000件→体型变大吃Boss）/ 守藏（安全带回100万灵石→行走仓库）/ 化龙（灵机75+撤离20次→龙角鳞片，吃东西降灵机翻倍）

**穷奇三路**：天冲（击杀500精英→永久飞行）/ 暴风（速通50局<10分钟→极速形态）/ 守望（救队友30次→侦察范围翻倍）

其余角色同理。**你的饕餮和别人的饕餮会进化成完全不同的样子。**

## 附录·混沌战斗系统——「无形拳」

### 核心：每局都不一样

混沌没有固定技能树。每局开始随机获得1个主动技能+1个被动特性（从其他7只异兽技能池随机抽取），但都是**"走样版"——它在模仿，但模仿得不太对**：

| 抽到的原版 | 混沌的走样版 | 效果差异 |
|-----------|------------|---------|
| 饕餮·暴食（吞噬投射物） | 混沌·吞（变球滚过去撞） | 不吃子弹，物理碾压，控制困难 |
| 穷奇·凌空突袭（飞行俯冲） | 混沌·跳（原地弹跳，落点随机） | 不能飞，弹跳极高+落地震荡 |
| 九尾狐·狐火阵（3团火） | 混沌·舞火（绕自己转圈甩不规则火球） | 数量更多但轨迹不可预测 |
| 白泽·万象扫描（揭示信息） | 混沌·凝视（朝一个方向"看"） | 没有信息功能，但被"看到"的敌人减速 |

**"这局混沌抽到啥了？！"——开盲盒的上瘾感。**

### 独有机制一：「无面」

混沌外观每局随机复制队友（联机）或上局遇到的某个敌人（单人）。开局不知道长什么样——进入地图后才"变"。联机时队友会看到两个"自己"在跑。被复制的人："它又变成我了！！"

### 独有机制二：「混沌律动」——舞蹈=蓄力

其他角色靠射击输出，**混沌靠跳舞**。独有「律动条」：

| 行为 | 律动值 | 说明 |
|------|--------|------|
| 贴脸闪避敌人攻击 | +15 | 最高效蓄力 |
| 队友在附近战斗 | +3/秒 | 它在学你们 |
| 原地不动不攻击不被打 | +5/秒 | 它在跳舞 |
| 被攻击 | -10 | 打断了舞步 |
| 主动攻击 | +1 | 它不太懂攻击 |

**律动条满→触发「混沌之舞」**：原地起舞3秒，周围敌人停止行动面朝混沌（它们在"看"），队友全属性+20%持续3秒，地面画出巨大墨色圆环（掩体+减速区）。

**最优玩法：不是冲上去砍，是在战场边缘闪避、观察、跳舞，关键时刻爆发一次扭转战局。**

### 独有机制三：「不可收割」

- 混沌灵机永远0。不涨不降
- 队友屏幕泼墨金裂时，混沌玩家看到的是安静水墨淡彩
- 收割者**看不见混沌**。队友被追杀时，混沌是唯一能安全行动的人→去开撤离点
- 代价：无法打开灵机裂隙（>50才出现的高价值物资）。安全但贫穷
- **收割者从混沌身边穿过→混沌站在原地跳舞→最魔性的画面**

### 独有机制四：「七窍」——被凿的伤疤=成长

混沌身上7个暗色疤痕。每完成条件点亮一窍：

| 窍 | 条件 | 效果 |
|----|------|------|
| 第一窍·视 | 用混沌完成10局 | 看到隐藏物品轮廓 |
| 第二窍·听 | 「混沌之舞」救场5次 | 听到队友血量<20%的心跳声 |
| 第三窍·嗅 | 发现10个暗线文档 | 闻到灵机裂隙方向 |
| 第四窍·味 | 复制饕餮技能时"吃"掉3种Boss攻击 | Boss伤害-30% |
| 第五窍·触 | 与每只同事角色都共舞过 | 复制技能走样程度降低 |
| 第六窍·思 | 找到实验室Σ全部终端记录 | 律动蓄力×1.5 |
| **第七窍·？** | 隐藏 | **凿开→混沌死了。永久。** |

**第七窍是陷阱。** 集齐6窍后游戏暗示"还有最后一窍"。触发时弹出庄子原文——"日凿一窍，七日而混沌死"。选"凿开"→混沌变成正常人形角色，固定技能，很强，但不再跳舞、不再复制、不再是混沌。选"离开"→什么都不发生，6窍，没有脸，还在跳舞。**这个选择会在社区引发暴风级讨论。**

# 第八章 打工人文化系统

## 考勤

每天有工时限制。可以加班（超时外勤）但第二天debuff"熬夜后遗症"。连续加班3天→"过劳"事件。**你可以永远不加班——但加班=更多收入=更快升级。这就是真实的陷阱。**

## 工单系统

替代传统任务。格式是IT工单：
- 日常（绿）："3号节点例行检修"
- 紧急（橙）："清明市集灵网中断，需现场排查"
- 机密（红）："广寒废站深层数据异常"
- 暗线（墨色，来源不明）

## 公司邮件

工位电脑收邮件。HR温馨提示 / 主管催周报 / 同事八卦 / 匿名暗线信。**邮件是叙事的主要载体。**

## 茶水间八卦

同事闲聊：搞笑的（"听说混沌又变成CEO在走廊溜达了"）+ 暗线线索。

## 公司今日热搜

工位屏幕显示程序生成的搞笑标题，偶尔混入暗线：
- "#饕餮又把茶水间吃了#"
- "#实验室Σ再次出现异常能量波动 公司否认与外勤失踪有关#"

## 办公室政治

同事关系网。你站谁的队→影响工单类型和情报来源。

## 死亡 = 内容（Hades式）

每次工伤回来，同事有独特反应。饕餮给你留饭。穷奇替你去看看出事的区域。白泽分析你的死因给出建议。九尾狐替你填保险单。**死得越多→跟同事关系越好→暗线推进。**

# 第九章 美术——六部作品精髓合成

## 参考作品基因提取

| 作品 | 它真正革命的是什么 | 提取的技法 |
|------|------------------|-----------|
| **雾山五行** | 战斗=书法——每一拳是一笔，溅出的不是血是墨 | 攻击=笔触、受击=墨溅、环境用皴擦法但会动 |
| **蜘蛛侠纵横宇宙** | 不同角色不同画风共存同帧；帧率本身是表演 | 8只异兽微妙画风差异；灵机高时帧率变化制造不安 |
| **深海** | 色彩爆发表达情绪——黑暗底色上炸开高饱和油画色团 | 灵机驱动色彩：安宁=水墨素雅，狂潮=墨底炸开金紫赤 |
| **界外狂潮** | 涂鸦质感粗犷线条+街头美学+混沌能量视觉 | Bug兽设计语言——不是生物，是系统错误的视觉化 |
| **遗忘之海** | 水面=记忆隐喻——光影穿透水层的层次感 | 灵脉=地下流动发光墨水，能透过地面看到 |
| **超自然行动组** | 日常生活×超自然力量的视觉反差 | 工位=正常办公画风，出外勤=泼墨爆发，**画风切换本身是体验** |

## 原则一：两个世界，两种画风

**工位（安全区）= 工笔淡彩 + 日常办公**
- 线条清晰、色调温暖、Q版角色在格子间里走动
- 像超自然行动组的"日常面"——让你放松，以为这就是搞笑办公游戏
- 细节：饕餮工位全是零食包装、穷奇椅子被翅膀刮坏、白泽桌上叠书、混沌角落里独自跳舞

**外勤（危险区）= 泼墨写意 + 赛博霓虹**
- 出门瞬间画风切换——线条变粗变飞白、背景从工笔变泼墨、色调从暖变冷
- **这个切换本身就是"上班/下矿"的心理开关**——你能"看到"危险

## 原则二：灵机浓度驱动画面演化

| 灵机 | 画面状态 | 视觉参考 |
|------|---------|---------|
| 0-24 安宁 | 水墨淡彩，留白多，山水清远，能看远处 | 雾山五行宁静山水 |
| 25-49 涌动 | 墨色加浓，边缘洇染，远景模糊，紫色灵脉光从地下透出 | 遗忘之海水下光层 |
| 50-74 狂潮 | **泼墨爆发**——屏幕像被甩了一碗墨，攻击墨迹特效变大变狂，霓虹色从裂缝渗出 | 深海色彩爆炸 |
| 75-89 溢界 | **帧率开始不稳定**——像纵横宇宙Miles紧张时的卡顿感。金色裂缝撕开天空。角色边缘"溶解"成墨滴 | 纵横宇宙帧率操控 |
| 90-100 灵爆 | 几乎全黑只剩墨与金。角色是发光墨迹，敌人是黑暗中金色轮廓。一幅末日水墨画 | 深海最深处+雾山极致泼墨 |

**玩家不需要看数字就知道灵机多高——画面本身在告诉你。**

## 原则三：每只异兽有"画风指纹"（纵横宇宙式）

| 角色 | 画风偏移 | 理由 |
|------|---------|------|
| 饕餮 | 线条最粗最圆润，着色偏暖偏饱和 | 贪吃、热情、物质感 |
| 穷奇 | 线条带飞白和速度线，着色偏冷锐 | 快、暴躁、风属性 |
| 白泽 | 线条最工整最细，微小文字标注浮在身边 | 知识分子、信息处理者 |
| 九尾狐 | 线条最流畅最"妖"，火焰用重彩 | 优雅、妖气、控制感 |
| 混沌 | **没有固定画风——画风在不断微妙变化**，像信号不稳 | 无形、无定、不属于任何分类 |

## 原则四：Bug兽 = 视觉故障（界外狂潮式）

Bug兽不是生物学设计，是**系统错误的具象化**：
- 逻辑兵：损坏的代码块长出了腿，身上乱码纹路
- 算法猎手：搜索算法可视化——多条追踪线汇聚成猎食形态
- 收割无人机：高维产物——几何形体，表面流动金色符文
- 变异体：正常怪被灵机污染，身上长出墨色"肿瘤"，不断滴墨

## 原则五：截图即传播

每个画面过检验：**发到社交媒体3秒内能让人停下滑动？**
- 战斗泼墨 → 壁纸级
- 灵爆全黑只剩金墨 → 史诗截图
- 混沌跳舞 → 全网破防图
- 工伤死亡 → 角色化为墨滴散开（美到不生气）
- 打卡撤离 → 画面从泼墨骤然回归淡彩（视觉上的"长舒一口气"）

## 三层视觉架构

- 底层：水墨废墟（皴擦法山石 + 工笔建筑 + 泼墨天空 + 赛博霓虹叠加）
- 中层：Q版3头身角色（墨线勾边 + 水墨着色 + 动作帧间墨滴飞溅）
- 顶层：灵力特效 = 泼墨爆发（墨汁在宣纸上炸开的效果）

**收割者降临 = 天空裂开，金墨从裂缝泄下，全画面变成一幅末日水墨。**

## UI = 打工

工牌HUD / 工具箱背包 / 手绘工程图地图 / 红头文件弹窗 / 员工手册设置菜单。

# 第十章 音乐音效——魔鬼在细节里

## 设计哲学

**音效是这个游戏的第二视觉。** 鸭科夫的鸭子叫声是传播利器——我们要做到每个音效都能成为表情包素材。

核心原则：**安静是默认，声音是事件。** 不铺满BGM，让每一个声音都有意义、有记忆点。

## 音乐系统：灵机驱动的动态分层

不是切换音轨，是**叠加/剥离乐器层**——玩家感受不到切换，但情绪被无缝引导：

| 灵机 | 乐器层 | 情绪 | BPM |
|------|--------|------|-----|
| 0-24 安宁 | 古筝泛音轻拨 + 风声 + 水滴 | 宁静、孤独 | 60 |
| 25-49 涌动 | +琵琶节奏型 + 低频底鼓（心跳节奏） | 警觉、紧张初现 | 80 |
| 50-74 狂潮 | +二胡旋律（不协和音程）+ 电子bass + 金属碰撞采样 | 危险、兴奋 | 110 |
| 75-89 溢界 | +唢呐（变调，诡异不是喜庆）+ 大鼓战鼓 + 心跳声盖过音乐 | 恐惧、肾上腺素 | 140 |
| 90-100 灵爆 | 所有乐器扭曲失调 → 逐渐消失 → **只剩心跳和低语** | 纯恐惧 | 心跳 |

**关键技术**：每个乐器层独立Loop，通过音量淡入淡出叠加。灵机变化时平滑过渡，不是硬切。Web Audio的GainNode可以做到。

### 工位音乐（完全不同的风格）

工位BGM = **不合时宜的轻松办公电梯曲**——用古筝和琵琶演奏的Muzak/Bossa Nova。
- 像是有人把中国古典乐器塞进了办公室背景音系统
- 偶尔广播插入："本公司灵脉维护率达99.7%，感谢各位外勤同事的付出。"
- **外勤回来听到这个BGM的那一刻→全身放松→这就是"回家"的感觉**

## 标志性音效设计（每一个都要成为记忆锚点）

### 打卡音效（全游戏最重要的声音）

**"哔——"**
- 不是普通的嘀声。是一个精心设计的复合音：
  - 基音：清脆的电子嘀声（800Hz正弦波，50ms）
  - 叠加：古筝单音"叮"（延迟20ms，模拟回响）
  - 尾音：轻微的"嗡——"共振（暗示灵机系统在确认你的数据）
  - **撤离成功时**：加一层柔和的和弦解决（紧张→释放的听觉句号）
  - **工伤时**：打卡声变成忙音"嘟——嘟——嘟——"+"您所拨打的外勤已离线"
- **这个声音必须让人听到就条件反射地舒一口气。** 就像黑魂篝火点燃的声音。

### 射击与打击音效

| 武器类型 | 射击声 | 命中声 | 设计理念 |
|---------|--------|--------|---------|
| 公司标配（低级枪） | 弹簧"啪嗒" | 软"噗" | 听起来就像办公用品→搞笑+弱 |
| 供应商T1-T2 | 气动"嗤——" | 清脆"啪" | 开始有力量感 |
| 供应商T3 | 低沉"嘭" + 金属回音 | "咔嚓"碎裂感 | 满足+力量 |
| 稀有/传说 | 雷鸣"轰"（二胡嘶吼采样叠加） | 大鼓"咚"+墨溅声 | 史诗感+水墨感 |
| 灵器 | 毛笔落纸"沙——"+ 水墨渗透声 | 墨汁泼洒"哗啦" | **射击=画画的听觉暗示** |

### 翻箱/搜刮音效（每个都不一样→上瘾）

搜刮不同容器有完全不同的声音，每个都带"惊喜感"：
- 工具箱：金属"咔哒"→翻找声→"叮铃哐啷"
- 抽屉：木质"嘎吱"→纸张翻动→偶尔"哗啦"掉出一堆东西
- 保险柜：厚重"嗡——咔嚓"开锁→沉重开门声→**发现好东西时古筝"叮～～"拉长音**
- 垃圾桶：塑料"啪嗒"→翻找声→偶尔"叮——"（垃圾桶里出好货→全游戏最搞笑的音效反差）
- 尸体/Bug兽残骸：湿润"噗嗤"→掏东西声→（这个就是要恶心一点）

### 公司广播（灵魂音效）

**平静女声，永远不合时宜地温馨：**

触发时机和具体台词：
- 灵机达到25："灵机浓度轻微上升。建议注意个人防护。祝您工作愉快。"
- 灵机达到50："灵机辐射已超标。请勿恐慌。恐慌会进一步提升灵机浓度。"
- 灵机达到75："收割预警已触发。请在60秒内撤离。如无法撤离——感谢您对公司的贡献。"
- 灵机达到90："……（沉默3秒）……您还在吗？"
- 灵爆100："（静电噪音）……它……来了。"
- 打开空箱子："很遗憾，该容器已被回收。下次运气会更好的。（语气完全不像在安慰）"
- 击杀精英怪："恭喜！您的绩效考核已记录。"
- 濒死存活："检测到您的生命体征异常。请问您需要叫救护车吗？拨打内线……算了，我们没有救护车。"
- 工伤（死亡）："外勤编号XXXX已离线。请同事们注意安全。以上。"

### 角色专属音效

每只异兽有独有的脚步声+受伤声+搜刮声：
- 饕餮：沉重"咚咚"步声 / 受伤"嗷呜"像被踩尾巴 / 搜刮时"嗅嗅嗅"闻味道
- 穷奇：快速"嗒嗒嗒"加翅膀"呼扑" / 受伤暴躁"切！" / 搜刮时"快快快"嘟囔
- 白泽：轻柔"哒哒"学者步 / 受伤"啊这——" / 搜刮时"有意思……"自言自语
- 九尾狐：优雅"踏踏"高跟步 / 受伤"哼——"不屑 / 搜刮时"脏死了"嫌弃但还是翻
- **混沌**：**没有脚步声。完全无声移动。** 它在你旁边你都不知道。唯一的声音是跳舞时身体发出的柔和"咚咚"——像心跳，又像鼓点。

### Bug兽音效（系统故障感）

- 逻辑兵：电子噪音"滋滋"巡逻声 + 发现你时"嘟嘟嘟"警报
- 算法猎手：搜索声"嗖嗖嗖"（像硬盘读取）+ 锁定时"叮——"
- 收割无人机：高频嗡鸣（让人生理不适）+ 接近时频率升高
- 变异体：原怪声音 + 叠加一层水声/墨液声"咕噜咕噜"
- 梼杌（伪装Boss）：**完全无声**——直到它攻击的一瞬间爆发出惊悚的尖叫

## 灵机声效层（恐惧制造机）

灵机升高时，**环境音开始不可信**：
- 灵机25+：偶尔出现方向不明的脚步声（50%是假的）
- 灵机50+：环境音开始回声/失真，像在水下
- 灵机75+：出现**倒放的声音**——你刚才的脚步声被倒着播放一遍
- 灵机90+：**所有声音消失。只剩心跳。和收割者的低语——听不清说什么，但你知道它在对你说话。**

## 制作方案

**全程程序化合成（Web Audio API）**，零外部音频文件：
- 基础波形：OscillatorNode（正弦/方波/锯齿/噪声）
- 滤波：BiquadFilterNode（低通/高通/带通）
- 包络：GainNode + 指数衰减
- 混响：ConvolverNode（程序化IR生成）
- 动态分层：多个AudioBufferSourceNode通过GainNode混合

优势：文件0KB、无限变体、实时响应灵机参数。劣势：需要精心调参。

# 第十章b 魔鬼细节——让人上头的100个小东西

## 手感（Game Feel）——Phase 0 生死线

### 打击感三件套

每次命中敌人，同时触发：
1. **Hitstop（顿帧）**：画面冻结2-4帧（33-66ms）。伤害越高冻越久。让玩家"感受到"重量
2. **屏幕震动**：方向朝攻击反方向，幅度2-6px。灵器攻击额外加旋转震动
3. **击退+墨溅**：敌人后退+受击点爆出墨迹粒子（大小/数量随伤害缩放）

### 移动手感

- 不是"按下=全速"。加速曲线0.08秒到满速，松手减速0.05秒（滑行感但不拖沓）
- 翻滚有1帧无敌（不是很长——鼓励精准而非滥用）
- 翻滚结束瞬间可以立即射击（取消后摇→操作上限高→高手爽）

### 射击手感

- 后坐力=屏幕微震+准星扩散+角色后退1-2px
- 换弹动画：角色做出明确的换弹动作，**换弹最后一帧"咔嚓"声→射击手感恢复→"满血复活"的爽感**
- 空弹匣继续按射击："咔咔咔"空击声→紧张感×10

## 微反馈（Juice）

| 事件 | 视觉 | 音效 | 触觉（手柄） |
|------|------|------|-------------|
| 暴击 | 大号墨溅+伤害数字放大弹出+0.05秒慢动作 | 重鼓"咚" | 强震 |
| 击杀 | 敌人化墨消散+经验数字飞向角色 | 古筝短音 | 轻震 |
| 捡稀有物品 | 金色光柱+物品图标放大旋转 | 古筝"叮～～"长音+琵琶琶音 | 持续轻震 |
| 灵机跨阶 | 全屏墨色波纹扩散 | 对应阶段乐器层进入 | 脉冲震 |
| 濒死存活 | 屏幕边缘血红→慢动作0.3秒→正常 | 心跳加速→重鼓一击 | 心跳震 |
| 撤离打卡 | 全屏从泼墨骤归淡彩→白光过渡 | "哔——"→和弦解决 | 长柔震 |

## 死亡不是惩罚，是内容

工伤（死亡）后：
- **死亡画面**：角色化为墨滴向四周散开→画面定格→变成一幅水墨画→"外勤报告"叠印上去
- **工伤统计**：存活时间 / 击杀数 / 搜刮价值 / 灵机峰值 / 死因（用公司公文体写："该外勤因灵机辐射超标导致——算了，就是浪。"）
- **回到工位**：同事独特反应（Hades式）。死法越离谱同事吐槽越狠
- **工伤保险**：保险柜设施等级决定回收比例。满级60%——这意味着满级后死亡痛感降低→敢浪→灵机更容易飙→更刺激→正向循环
